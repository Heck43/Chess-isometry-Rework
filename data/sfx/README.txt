ЗВУКОВЫЕ ЭФФЕКТЫ ДЛЯ CHESS ISOMETRY REWORK
==========================================

Поместите ваши звуковые файлы в эту папку для автоматической загрузки.

ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ:
- .wav (рекомендуется)
- .ogg 
- .mp3

НАЗВАНИЯ ФАЙЛОВ:
- hit.wav - звук при взятии фигуры (уже добавлен)
- move.wav - звук обычного хода
- check.wav - звук шаха
- checkmate.wav - звук мата
- select.wav - звук выбора фигуры

ПРИМЕЧАНИЯ:
- Имя файла (без расширения) используется как идентификатор звука
- Регистр не важен (Capture.wav = capture.wav)
- Громкость регулируется через настройки игры
- Файлы загружаются при запуске игры

Поместите ваш подготовленный wav файл для взятия фигур в эту папку!
